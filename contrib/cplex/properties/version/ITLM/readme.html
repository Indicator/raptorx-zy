IBM License Metric Tool Signature Files

Introduction

IBM License Metric Tool (ILMT) helps Passport Advantage clients determine their full and sub-capacity PVU licensing requirements.

Learn more at http://www.ibm.com/software/tivoli/products/license-metric-tool

As IBM ILOG CPLEX Optimization Studio Edition components are embedded within the customer's application, there is no way for IBM to install a signature file (*.swtag) with each deployed application. Consequently, customers who want ILMT to generate reports must copy the signature file for the deployment license with each application that they deploy.

Steps to enable ILMT

  1. Ensure that your company is properly entitled.
  2. Open the properties/version/Tags.zip file in your installation directory that contains the signature for the deployment license metric.
  3. Extract the signature file (*.swtag).
  4. Copy the signature file to each disk where your application is installed. The recommended method is to add the signature file to your IBM ILOG CPLEX Optimization Studio application so that the signature file is deployed as part of the application.
